    --patch '{"data":{"mesh":"extensionProviders:\n- name: {{ include "hmac-manager.fullname" . }}\n  envoyExtAuthzHttp:\n    service: {{ include "hmac-manager.fullname" . }}.{{ .Values.namespace }}.svc.cluster.local\n    port: {{ .Values.service.port }}\n    withRequestBody:\n      maxRequestBytes: {{ .Values.istio.meshConfig.maxRequestBytes }}\n      allowPartialMessage: {{ .Values.istio.meshConfig.allowPartialMessage }}"}}'

This patches the existing Istio ConfigMap rather than replacing it.
Run this once after install (and again after any Istio upgrade that resets the ConfigMap).
